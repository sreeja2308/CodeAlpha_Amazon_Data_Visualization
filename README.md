# Amazon Products Data Visualization

This project focuses on visualizing and analyzing Amazon product data using Python. Various charts and graphs were created to understand product categories, ratings, discounts, and pricing trends.

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Visualizations Included
- Top Product Categories
- Ratings Distribution
- Discount Percentage Analysis
- Actual Price vs Discounted Price

## Key Insights
- Electronics products dominate Amazon listings.
- Most products have high customer ratings.
- Significant discounts are common across products.
- Higher actual prices generally lead to higher discounted prices.

## Dataset
Amazon Products Dataset from Kaggle.
